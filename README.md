# 🤖 Telegram Бот — Первый проект

Простой Telegram-бот на Python с использованием библиотеки `pyTelegramBotAPI`. Бот умеет отвечать на сообщения, приветствовать пользователя, обрабатывать фотографии и поддерживает инлайн-кнопки.

## 🚀 Функционал

| Команда / Действие | Что делает |
|-------------------|-------------|
| `/start` | Отправляет приветственное сообщение с фото и показывает клавиатуру с кнопками |
| `/main` или `/hello` | Приветствует пользователя по имени |
| `/help` | Выводит справочную информацию (HTML-форматирование) |
| `Привет` (текст) | Бот отвечает дружелюбным приветствием |
| `ID` (текст) | Бот отвечает с ID пользователя |
| Отправка фото | Бот хвалит фото и предлагает кнопки: «Перейти на сайт», «Удалить фото», «Изменить текст» |
| Кнопка «Удалить фото» | Удаляет отправленное фото |
| Кнопка «Изменить текст» | Изменяет текст сообщения |

## 🛠 Технологии

- **Python** — язык программирования
- **pyTelegramBotAPI** — библиотека для работы с Telegram Bot API
- **Telegram Bot API** — взаимодействие с ботом
